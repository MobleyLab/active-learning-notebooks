# active-learning-notebooks
Jupyter notebooks for ligand optimization studies involving active learning

### Install Python environment
```bash
conda env create --name active-learning --file environment.yml
```

### Publications:
- MultiobjectiveAL.ipynb: [A. Kubincova and D. Mobley, A simple compound prioritization method for drug discovery considering multi-target binding, Digit. Discov. 2026, 5, 1204-1214.](https://doi.org/10.1039/D5DD00464K)
- DockingMLEnsemble.ipynb: [A. Kubincova and D. Mobley, Bridging between structure-based and data-driven affinity prediction, ChemRxiv 2026 (preprint)](https://doi.org/10.26434/chemrxiv.15000463/v1)
